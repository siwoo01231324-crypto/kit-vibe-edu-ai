# kit-vibe-edu-ai (루트)

## 목적
AI 교육 플랫폼 MVP. 교사가 퀴즈를 생성하고 학생이 실시간으로 참여하며, Claude AI가 이해도를 분석하는 Next.js 15 웹 애플리케이션.

## 기술 스택
- **프레임워크**: Next.js 15.2.x (App Router) + React 19 + TypeScript 5 strict
- **스타일**: Tailwind CSS v3 + shadcn/ui
- **DB/Auth/Realtime**: Supabase (PostgreSQL + Auth + Realtime)
- **AI**: Anthropic Claude API (claude-sonnet-4-6)
- **테스트**: Vitest 2 (단위/통합) + Playwright 1 (E2E)

## 구조
```
kit-vibe-edu-ai/
├── apps/
│   └── web/        ← Next.js 앱 단독 (모든 소스·설정·의존성)
├── docs/
├── scripts/
└── ...
```

| 경로 | 역할 |
|------|------|
| `CLAUDE.md` | 아키텍처 불변식·규칙·작업 흐름 (세션 시작 시 필독) |
| `AGENTS.md` | 레포 전체 목차 |
| `apps/web/` | Next.js 15 앱 — 소스·설정·의존성 모두 이 안에 |
| `apps/web/src/` | 모든 애플리케이션 소스 코드 (`src/app/`, `src/components/`, `src/lib/`, `src/types/`) |
| `apps/web/tests/` | 테스트 루트 (`unit/`, `integration/`, `e2e/`) |
| `docs/` | 프로젝트 문서 SOT |
| `scripts/` | 불변식 검증 등 유틸리티 |

## 핵심 워크플로우
```
/bi → /si → /plan → /fi → /ci
```

## 주의
- 커밋·푸시 전 항상 사용자 확인 필요 (CLAUDE.md 행동 규칙)
- 모든 디렉토리에 .ai.md 포함 필수
- Next.js 앱은 `apps/web/` 안에서 관리: `cd apps/web && npm run dev`
- `@/*` import alias 는 `apps/web/src/` 기준 (`./src/*`)
