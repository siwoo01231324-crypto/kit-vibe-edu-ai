# apps/web — Next.js 15 애플리케이션

> kit-vibe-edu-ai 의 단일 Next.js 15 앱. 모든 애플리케이션 코드, 설정, 의존성이 이 디렉토리에 존재한다.

## 목적
- Next.js 15 App Router 기반 학생/교사 대시보드
- TypeScript strict + Tailwind v3 + Vitest + Playwright

## 디렉토리 구조
- `src/app/` — App Router 페이지/레이아웃/API
- `src/components/` — React 컴포넌트
- `src/lib/` — 도메인 로직, Supabase 클라이언트, 프롬프트
- `src/types/` — 타입 정의
- `tests/` — Vitest 단위/통합 + Playwright E2E
- 루트 설정 파일들 (next.config.ts, tsconfig.json, package.json 등)

## 실행
```bash
cd apps/web && npm run dev
```

## 규칙
- 모든 Next.js 관련 파일은 이 디렉토리 안에서 관리
- 프로젝트 문서는 `../../docs/` (레포 루트)
- 인프라 스크립트는 `../../scripts/`
