# apps/web/tests/

## 목적
프로젝트 테스트 루트. `apps/web/src/` 외부에 위치하며 단위·통합·E2E 테스트를 포함한다.

## 구조
| 디렉토리 | 도구 | 역할 |
|----------|------|------|
| `unit/` | Vitest | 순수 함수 단위 테스트 |
| `integration/` | Vitest + Supabase 로컬 | API Route → DB 통합 테스트 |
| `e2e/` | Playwright | 전체 흐름 E2E 시나리오 |
| `setup.ts` | Vitest setupFiles | @testing-library/jest-dom matchers 등록 |

## 실행 명령어
```bash
cd apps/web && npm run test          # Vitest 단위+통합 (once)
cd apps/web && npm run test:watch    # Vitest watch 모드
cd apps/web && npm run e2e           # Playwright E2E (webServer 자동 기동)
```

## 규칙
- `apps/web/tests/` 경로는 `apps/web/src/` 외부 유지
- E2E 테스트는 `apps/web/tests/e2e/` 에만 위치
- Vitest 대상: `apps/web/tests/unit/**`, `apps/web/tests/integration/**`
