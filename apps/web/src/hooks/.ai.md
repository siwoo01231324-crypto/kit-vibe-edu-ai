# src/hooks

## 목적
클라이언트 컴포넌트에서 사용하는 React 훅을 모아둔다.

## 파일 구조
- `useQuestions.ts` — 퀴즈 문항 CRUD 훅 (fetch, add, update, delete, moveUp, moveDown)

## 주요 규칙
- 모든 훅은 'use client' 디렉티브 없이 작성 (훅 파일 자체는 서버 컴포넌트와 충돌하지 않음)
- Supabase 직접 호출 (API route 경유 없음)
- optimistic update 대신 서버 응답 후 상태 반영
