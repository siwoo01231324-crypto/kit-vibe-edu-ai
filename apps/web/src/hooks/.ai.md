# src/hooks

## 목적
클라이언트 컴포넌트에서 사용하는 React 커스텀 훅을 모아둔다.

## 파일 구조
- `useQuestions.ts` — 퀴즈 문항 CRUD 훅 (fetch, add, update, delete, moveUp, moveDown)
- `useSessionStatus.ts` — Supabase Realtime으로 sessions 테이블의 status 변화를 구독. 대기 화면에서 사용.

## 주요 규칙
- 모든 훅은 `'use client'` 지시어 필요
- Supabase 직접 호출 (API route 경유 없음)
- Supabase 구독은 unmount 시 반드시 `removeChannel` 정리
- optimistic update 대신 서버 응답 후 상태 반영
