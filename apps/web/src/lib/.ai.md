# src/lib/

## 목적
비즈니스 로직·유틸리티·외부 SDK 클라이언트. UI 독립적인 순수 함수와 서비스 레이어를 포함한다.

## 구조
| 경로 | 역할 |
|------|------|
| `supabase/` | Supabase 클라이언트 (client.ts, server.ts, middleware.ts) |
| `scoring.ts` | 점수 계산 로직 — `calculateScore(isCorrect, responseTimeMs)` → 0 또는 100~1000 |
| `validation.ts` | 입력 검증 유틸 — `validateNickname(nickname)` → boolean (2~12자, 한글/영문/숫자/밑줄) |
| `join-code.ts` | join_code 생성 유틸 (예정) |
| `prompts/` | Claude AI 프롬프트 템플릿 (insights.ts, class-draft.ts, 예정) |

## 규칙
- 순수 함수 우선 (사이드 이펙트 최소화)
- Supabase 클라이언트는 `supabase/client.ts` (브라우저), `supabase/server.ts` (서버) 분리
- Claude API 클라이언트는 서버 전용 (API Key 노출 금지)
