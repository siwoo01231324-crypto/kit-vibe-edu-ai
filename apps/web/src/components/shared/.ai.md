## 목적
특정 도메인에 종속되지 않고 여러 페이지/기능에서 재사용 가능한 공유 UI 컴포넌트 모음.

## 파일 구조
- `QRCodeDisplay.tsx` — QR 코드 이미지 생성 및 PNG 다운로드 버튼 (Client Component, `qrcode` 패키지 사용)

## 주요 규칙
- 모든 컴포넌트는 `'use client'` 지시자 필수 (서버 컴포넌트에서 import 금지)
- `QRCodeDisplay`: `useEffect` 내에서 `QRCode.toDataURL()` 비동기 호출 (hydration mismatch 방지)
- 다운로드 파일명 고정: `session-qr.png`
