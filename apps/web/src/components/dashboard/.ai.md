## 컴포넌트

### InsightPanel
**Props:** `{ insights: InsightResult }`
AI가 생성한 인사이트(취약 개념 / 강점 개념 / 다음 수업 포커스)를 카드 형태로 시각화.

### ResponseChart
**Props:** `{ questions: Question[]; responses: Response[] }`
groupByQuestion() 호출 → 문항별 정답률 + 4개 선택지 분포를 CSS flex div로 렌더.
Chart.js/Recharts 금지.

### SessionSidebar
**Props:** `{ sessions: SessionRow[]; selectedId?: string }`
세션 목록 + status 필터 (all/draft/active/ended). 클릭 시 `/teacher/dashboard?session={id}` 이동. selectedId 세션은 orange highlight.

### SessionDetailClient
**Props:** `{ session, questions, initialResponses, thumbsFeedbacks? }`
useRealtimeResponses로 실시간 갱신. 참여 인원 뱃지 + 학생 피드백 패널(thumbsFeedbacks) + ResponseChart 렌더.
모든 뷰 링크는 `/teacher/dashboard?session=<id>&view=<view>` 쿼리파라미터 방식.

### ClassDraftPanel
**Props:** `{ content: string; sourceSessionId: string }`
마크다운 수업 초안 표시 + "AI 세션 생성" 버튼 → DraftSessionConfirmModal 연동.

### DraftSessionConfirmModal
**Props:** `{ sourceSessionId, onClose }`
preview API 호출로 AI 생성 문항 미리보기 → 확인 시 from-draft API로 새 세션 생성.

### ThumbsFeedbackPanel
Async server component (deprecated — 현재 SessionDetailClient에 인라인으로 통합됨).
