## 컴포넌트

### InsightPanel
**Props:** `{ insights: InsightResult }`
AI가 생성한 인사이트(취약 개념 / 강점 개념 / 다음 수업 포커스)를 카드 형태로 시각화.

### ResponseChart
**Props:** `{ questions: Question[]; responses: Response[] }`
groupByQuestion() 호출 → 문항별 정답률 + 4개 선택지 분포를 CSS flex div로 렌더.
Chart.js/Recharts 금지.

### SessionSidebar
**Props:** `{ sessions: SessionRow[] }`
세션 목록 + status 필터 (all/draft/active/ended). 클릭 시 /teacher/sessions/{id} 이동.

### SessionDetailClient
**Props:** `{ session, questions, initialResponses }`
useRealtimeResponses로 실시간 갱신. 참여 인원 뱃지 + ResponseChart 렌더.
InsightPanel 슬롯은 #37 후속 작업에서 연결.
