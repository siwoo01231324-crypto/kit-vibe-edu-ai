# src/components/

## 목적
재사용 UI 컴포넌트. 페이지 간 공유되는 시각 요소를 포함한다.

## 구조 (예정)
| 디렉토리 | 역할 |
|----------|------|
| `ui/` | shadcn/ui 기반 기본 컴포넌트 (Button, Card, Input 등) |
| `quiz/` | 퀴즈 게임 전용 컴포넌트 (QuestionCard, AnswerOptions, Leaderboard, Timer) |
| `dashboard/` | 교사 대시보드 컴포넌트 (SessionCard, ResponseChart, InsightPanel) |
| `shared/` | 공통 유틸 컴포넌트 (QRCodeDisplay, LoadingSpinner) |

## 규칙
- Client Component 는 `'use client'` 명시 필수
- shadcn/ui 컴포넌트는 `ui/` 에 위치
- 컴포넌트 파일명은 PascalCase
