# (student) route group — .ai.md

## 목적
익명 학생 사용자를 위한 라우트 그룹. Supabase Auth 없이 sessionStorage 기반으로 동작.

## 구조
```
(student)/
├── layout.tsx              — 인증 불필요, 빈 래퍼 레이아웃
├── join/
│   ├── page.tsx            — join_code + 닉네임 입력 폼 (직접 접속)
│   └── [code]/
│       └── page.tsx        — QR 스캔 URL 진입 (code pre-filled, 닉네임만 입력)
└── waiting/
    └── [sessionId]/
        └── page.tsx        — 대기 화면, Realtime 구독으로 status 변화 감지
```

## 역할
- `/join` — 학생이 6자리 join_code와 닉네임을 입력해 세션 참여
- `/join/[code]` — QR 코드 스캔 시 코드가 URL에 포함되어 닉네임만 입력
- `/waiting/[sessionId]` — 대기 화면. 세션 status가 'active'로 바뀌면 `/quiz/[sessionId]`로 이동

## 상태 관리
- 학생 정보: `sessionStorage['studentSession']` = `{ sessionId, nickname }`
- Realtime: `useSessionStatus` 훅 사용 (src/hooks/useSessionStatus.ts)
