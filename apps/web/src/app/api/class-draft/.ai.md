# apps/web/src/app/api/class-draft

## 목적
수업 초안 생성 API 엔드포인트 모음.

## 구조
```
class-draft/
└── generate/
    └── route.ts   # POST /api/class-draft/generate
```

## 역할
- `generate/route.ts`: 인사이트 기반 마크다운 수업 초안 생성
  - Auth 체크 (401)
  - session_id 소유권 검증 (403)
  - ai_insights 존재 확인 (400 NO_INSIGHTS)
  - class_drafts 캐시 히트 시 즉시 반환
  - buildDraftPrompt → Claude API 호출 (maxTokens: 2048)
  - class_drafts INSERT 후 { content } 반환

## 의존성
- `@/lib/anthropic`: callClaude
- `@/lib/prompts/class-draft`: buildDraftPrompt
- `@/lib/supabase/server`: createClient
- DB: sessions, ai_insights, class_drafts 테이블
