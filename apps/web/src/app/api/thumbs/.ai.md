## 목적
인증되지 않은 학생이 thumbs_feedback 테이블에 INSERT할 수 있도록 service role key를 사용해 RLS를 우회하는 API 라우트.

## 파일
- `route.ts` — POST `/api/thumbs`

## 요청 스키마
```json
{ "session_id": "uuid", "nickname": "string", "thumb": "up|down", "comment": "string|null" }
```

## 왜 API 라우트인가?
학생은 Supabase Auth 없이 동작(anonymous). 직접 Supabase INSERT는 RLS에 막힘.
`createAdminClient()` (service role key)는 서버사이드에서만 안전하게 사용 가능.
