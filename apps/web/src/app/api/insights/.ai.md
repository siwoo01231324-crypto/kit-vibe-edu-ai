## 엔드포인트

### POST /api/insights/generate

**Body:** `{ session_id: string (uuid) }`

**응답:**
- 200: InsightResult (top_weak_concepts, strong_concepts, next_class_focus)
- 400: VALIDATION_ERROR
- 401: UNAUTHORIZED
- 403: FORBIDDEN (타인 세션)
- 500: CLAUDE_ERROR | PARSE_ERROR

**캐싱:** ai_insights 테이블 기반 DB 캐시. 동일 session_id는 Claude 미호출.
