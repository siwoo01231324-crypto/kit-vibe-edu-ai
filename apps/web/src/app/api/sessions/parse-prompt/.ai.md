# api/sessions/parse-prompt

## 목적
선생님이 입력한 자연어(예: "고1 수학, 피타고라스 정리 퀴즈 5문제")를 Claude API로 파싱하여 세션 제목·과목·학년을 구조화된 JSON으로 반환한다.

## 파일 구조
- `route.ts` — POST 핸들러: Supabase 인증 → Zod 입력 검증 → callClaude → parseSessionParseResponse → JSON 응답

## API 스펙
- **메서드**: POST
- **인증**: Supabase 세션 필요 (401 반환)
- **요청 body**: `{ prompt: string }` (10~2000자)
- **응답**: `{ title, subject, grade, missing: string[] }`
- **에러**: 400 VALIDATION_ERROR (입력 검증), 400 PARSE_ERROR (AI 파싱 실패), 500 API_ERROR (Claude 호출 실패)

## 연관 모듈
- `lib/prompts/session-parse.ts` — 프롬프트 빌더 + 파서
- `lib/anthropic.ts` — callClaude() (MOCK_CLAUDE=1 지원)
