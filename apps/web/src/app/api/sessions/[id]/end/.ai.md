## 목적
active 상태의 세션을 ended 상태로 전이시키는 POST API.

## 파일 구조
- `route.ts` — POST /api/sessions/[id]/end 핸들러

## 주요 규칙
- 소유권 가드: `.eq('teacher_id', user.id)` 필수 (RLS 이중 방어)
- 상태 전이 가드: `.eq('status', 'active')` — active → ended 전이만 허용
- 전이 불가 시 409 INVALID_TRANSITION 반환
- ended_at 타임스탬프 기록
