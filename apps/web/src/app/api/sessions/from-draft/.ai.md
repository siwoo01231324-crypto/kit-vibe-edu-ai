## 목적
교사가 생성한 수업 초안을 바탕으로 Claude가 퀴즈 문항을 자동 생성하고, 새 세션과 문항을 DB에 저장.

## 파일 구조
- `route.ts` — POST `/api/sessions/from-draft` — 새 세션 생성 + 문항 bulk insert
- `preview/route.ts` — POST `/api/sessions/from-draft/preview` — 문항 미리보기 (DB 저장 없음)

## 흐름
1. `preview` 호출 → Claude haiku tool_use → 제목 + 문항 3~5개 반환
2. 교사가 미리보기 확인 후 확인 → `from-draft` 호출 → sessions INSERT + questions bulk INSERT

## Claude 연동
- 모델: `claude-haiku-4-5-20251001`
- Tool: `generate_session_questions` (DRAFT_QUESTIONS_TOOL in lib/prompts/draft-questions.ts)
- 입력: 수업 초안 content + 취약 개념(top_weak_concepts from ai_insights)
