# app/api/sessions

## 목적
교사가 새 수업 세션을 생성하는 POST 엔드포인트를 제공한다.

## 파일 구조
- `route.ts` — POST /api/sessions 핸들러

## 주요 규칙
- Supabase 인증 확인 후 미인증 시 401 반환
- zod로 body 검증 (title, subject, grade 필수)
- join_code는 generateUniqueJoinCode()로 생성
- 성공 시 { id, join_code } 반환 (201)
- 오류 형식: { error: 'UNAUTHORIZED' | 'VALIDATION_ERROR' | 'DB_ERROR', details? }
