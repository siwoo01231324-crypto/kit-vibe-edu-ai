## 목적
교사가 Google OAuth로 로그인하는 진입점.

## 구조
- `page.tsx` — 서버 컴포넌트, searchParams에서 next/error 추출
- `GoogleSignInButton.tsx` — 클라이언트 컴포넌트, signInWithOAuth 호출

## 동작
1. 버튼 클릭 → supabase.auth.signInWithOAuth({ provider: 'google' })
2. Google 동의 → /auth/callback?code=...&next=... 으로 귀환
3. 로그인 실패 시 ?error= 파라미터로 메시지 표시

## 관련 파일
- `/auth/callback/route.ts` — OAuth 코드 교환
- `src/lib/supabase/client.ts` — 브라우저 Supabase 클라이언트
