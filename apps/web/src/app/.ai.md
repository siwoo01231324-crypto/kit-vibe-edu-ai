# src/app/

## 목적
Next.js 15 App Router 루트. 페이지·레이아웃·API Routes 를 포함한다.

## 구조
| 경로 | 역할 |
|------|------|
| `(student)/` | 학생 퀴즈 게임 페이지 (join, quiz) — 레이아웃 분리 |
| `(teacher)/` | 교사 대시보드 페이지 (dashboard, sessions) — 인증 필요 |
| `api/` | API Routes (sessions, insights, class-draft) |
| `globals.css` | Tailwind base/components/utilities |
| `layout.tsx` | 루트 레이아웃 (`<html lang="ko">`), 앱명 **Seeya** 메타 태그 |
| `page.tsx` | 랜딩 페이지 — h1 **Seeya!**, 태그라인 "교실 모두의 시야를 넓혀주는 AI 파트너" |

## 규칙
- Server Components 기본; Client Components 는 `'use client'` 명시
- Route Groups `(student)`, `(teacher)` 로 레이아웃 분리
- API Routes 는 `route.ts` 파일 사용
