## 목적
특정 세션의 AI 인사이트를 표시하고 수업 초안 생성을 트리거하는 영역.

## 파일 구조
- `page.tsx` — 독립 페이지 진입점 (params 언래핑 후 InsightsContent 렌더)
- `InsightsContent.tsx` — Client Component. 독립 페이지 및 대시보드 인라인 양쪽에서 재사용 가능

## InsightsContent 동작
1. 마운트 시 `ai_insights` 테이블에서 기존 인사이트 로드
2. "인사이트 생성/재생성" → `/api/insights/generate` POST
3. "수업 초안 생성" → `/api/class-draft/generate` POST → draftRef로 자동 스크롤
4. 수업 초안 생성 후 ClassDraftPanel 렌더 (AI 세션 생성 포함)
