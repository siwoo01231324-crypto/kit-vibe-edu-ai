## 목적
세션 시작/종료 버튼, 참여 코드, QR 코드를 제공하는 교사 전용 라이브 세션 화면.

## 파일 구조
- `page.tsx` — 서버 컴포넌트: 세션 데이터 조회 + joinUrl 생성 후 LiveSessionClient에 전달
- `LiveSessionClient.tsx` — 클라이언트 컴포넌트: activate/end API 호출 및 QRCodeDisplay 렌더링

## 주요 규칙
- `page.tsx`는 서버 컴포넌트; `LiveSessionClient.tsx`는 `'use client'`
- NEXT_PUBLIC_BASE_URL 환경변수로 joinUrl 생성 (`/join/{join_code}`)
- 세션 상태(draft/active/ended)에 따라 버튼 분기 렌더링
- activate/end 성공 후 `router.refresh()`로 서버 컴포넌트 재실행
