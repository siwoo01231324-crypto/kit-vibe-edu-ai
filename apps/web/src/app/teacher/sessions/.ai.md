# app/teacher/sessions

## 목적
교사가 수업 세션을 생성·편집하고 실시간 응답 현황을 볼 수 있는 페이지들을 포함한다.

## 파일 구조
- `new/page.tsx` — 세션 생성 폼 (자연어 textarea + "AI로 채우기" 버튼 → POST /api/sessions/parse-prompt → 폼 자동 채움; 기존 수동 입력 폼 하위 호환 유지 → POST /api/sessions → /teacher/sessions/[id]/edit 리다이렉트)
- `[id]/page.tsx` — 세션 상세 Server Component (소유권 검증 + 초기 데이터 fetch → SessionDetailClient 주입) — #31
- `[id]/edit/page.tsx` — 세션 편집 서버 컴포넌트 (소유권 확인 후 QuestionEditor 렌더링)
- `[id]/edit/QuestionEditor.tsx` — 퀴즈 문항 CRUD 클라이언트 컴포넌트 (useQuestions 훅 사용)

## 라우팅
- `/teacher/sessions/new` → 새 세션 만들기
- `/teacher/sessions/[id]` → 세션 실시간 응답 현황 대시보드
- `/teacher/sessions/[id]/edit` → 세션 편집 (문항 추가/수정/삭제/순서변경)
- `/teacher/sessions/[id]/insights` → AI 인사이트 + 수업 초안 생성 (마크다운 미리보기 + 복사)
