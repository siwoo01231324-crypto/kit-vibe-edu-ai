# app/teacher/sessions

## 목적
교사가 수업 세션을 생성하고 편집할 수 있는 페이지들을 포함한다.

## 파일 구조
- `new/page.tsx` — 세션 생성 폼 (제목/과목/학년 입력 → POST /api/sessions → /teacher/sessions/[id]/edit 리다이렉트)
- `[id]/edit/page.tsx` — 세션 편집 + 퀴즈 문항 CRUD (Issue #29에서 구현)

## 라우팅
- `/teacher/sessions/new` → 새 세션 만들기
- `/teacher/sessions/[id]/edit` → 세션 편집 (문항 추가/수정/삭제)
