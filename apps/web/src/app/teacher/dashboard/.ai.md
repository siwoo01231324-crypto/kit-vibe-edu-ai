## 목적
로그인한 교사의 홈 화면. 본인 세션 목록을 사이드바로 표시하고 선택 세션의 상세/편집/라이브/인사이트를 우측 영역에 인라인 렌더.

## 파일 구조
- `page.tsx` — Server Component. `?session=<id>&view=<view>` 쿼리 파라미터로 뷰 분기

## 뷰 분기 (`view` 쿼리 파라미터)
| view | 렌더 컴포넌트 |
|------|-------------|
| (없음) | SessionDetailClient — 실시간 응답 현황 + 학생 피드백 |
| `edit` | QuestionEditor |
| `live` | LiveSessionClient |
| `insights` | InsightsContent |

## SPA 패턴 (#56)
- 모든 뷰 전환은 `<Link href="/teacher/dashboard?session=<id>&view=<view>">` 로 처리
- 페이지 이동 없이 사이드바 항상 유지
