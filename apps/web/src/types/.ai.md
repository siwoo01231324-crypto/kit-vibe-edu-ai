# src/types/

## 목적
프로젝트 전역 TypeScript 타입 정의. 소스 코드를 포함하지 않으며 타입만 export 한다.

## 구조 (예정)
| 파일 | 역할 |
|------|------|
| `database.ts` | Supabase generated types (DB 스키마 기반) |
| `api.ts` | API Request/Response 인터페이스 |
| `domain.ts` | 도메인 모델 타입 (Session, Question, Response 등) |

## 규칙
- 런타임 코드 금지 — 타입 선언만 허용
- Supabase 타입은 `supabase gen types typescript` 로 자동 생성
