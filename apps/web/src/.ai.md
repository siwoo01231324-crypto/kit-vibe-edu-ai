# apps/web/src/

## 목적
모든 애플리케이션 소스 코드의 루트 디렉토리. Next.js 15 App Router 패턴을 따른다.
프로젝트 루트는 `apps/web/` 이며, `tests/` 는 `apps/web/tests/` 에 위치한다.

## 구조
| 경로 | 역할 |
|------|------|
| `apps/web/src/app/` | Next.js App Router — 페이지·레이아웃·API Routes |
| `apps/web/src/components/` | 재사용 UI 컴포넌트 |
| `apps/web/src/lib/` | 비즈니스 로직·유틸리티·외부 SDK 클라이언트 |
| `apps/web/src/types/` | TypeScript 타입 정의 (DB generated, API, 도메인) |
| `apps/web/src/middleware.ts` | Next.js 미들웨어 (Auth 라우팅) |

## 규칙
- 모든 import alias `@/*` 는 `./src/*` 를 가리킨다 (`apps/web/tsconfig.json` paths 기준)
- Server Components 기본; Client Components 는 파일 상단 `'use client'` 명시
- 각 서브 디렉토리에 `.ai.md` 포함 필수
