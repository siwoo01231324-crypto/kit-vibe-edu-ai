# docs/branding/ — 브랜딩 전략 문서

## 목적
**Seeya(시야)** 프로젝트의 실무 수준 브랜딩 전략을 담는 디렉토리.
글로벌/국내 EdTech 브랜딩 리서치, 네이밍 전략, 브랜드 아이덴티티, 후킹 포인트, 톤앤매너 가이드를 포함한다.

## 구조

| 파일 | 역할 |

## 참조 관계

- **출력 소비처**: 발표 슬라이드, UI 디자인 시스템, 마케팅 카피, 데모 시나리오
- **관련 이슈**: #17

## 규칙
- 모든 브랜딩 주장에 출처 명시 (팩트 기반, 허구 금지)
- 미존재 파일(project-plan.md, narrative-spine.md, research-supplement.md) 인용 금지
- 컬러 팔레트는 초안 방향(Draft Direction)이며 경쟁사 분석에 따라 조정 가능
- "관계 회복"은 상위 포지셔닝(Why)으로만 사용, 기능 설명(What)으로 혼동 금지
