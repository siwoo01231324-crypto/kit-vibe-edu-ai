# docs/demo/ — 공모전 데모 자산

## 목적

공모전 제출용 라이브 데모를 위한 스크립트, 배포 가이드, 폴백 자료를 관리합니다.

## 구조

| 파일 | 역할 |
|------|------|
| `demo-script.md` | 3분 데모 타임라인 + 단계별 멘트 + 폴백 시나리오 |
| `manual-deploy-guide.md` | Supabase 원격 + Vercel 배포 사용자 수동 가이드 |

## 역할

- 데모 재현성 보장: 누구든 이 디렉토리의 문서만으로 동일한 데모를 진행할 수 있어야 합니다
- 배포 절차 문서화: 환경변수, OAuth, seed.sql 실행 순서를 명시합니다
- 백업 경로 제공: 네트워크·AI 장애 시 대응 절차를 포함합니다

## 편집 규칙

- 영상 바이너리(`.mp4`, `.mov` 등) 커밋 금지 — 레포 규칙 (5MB 초과 + .gitignore)
- `demo-script.md` 수정 시 인간 검토 서명란을 유지하세요 (불변식 2)
- 실제 API 키, UUID 등 시크릿을 이 디렉토리에 커밋하지 마세요
