## 예상 Q&A

**Q: 실시간 응답은 어떻게 작동하나요?**  
A: Supabase Realtime 구독을 사용합니다. 학생이 응답을 제출하면 교사 화면에 즉시 반영됩니다.

**Q: AI는 어떤 모델을 쓰나요?**  
A: Anthropic Claude Haiku를 사용합니다. 빠른 응답과 비용 효율성을 위해 선택했습니다.

**Q: 데이터는 안전한가요?**  
A: Supabase Row Level Security(RLS)로 교사는 자신의 세션 데이터만 접근할 수 있습니다.
